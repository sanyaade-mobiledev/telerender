PushContent
===========

Push media content to a remote device with a popup menu.


Usage
-----

Right click a media element on the current web page. The "Push Content" popup menu will send the URI of that element
to all media renderers known on the network. 


Install
-------

  * Edit the pushcontent@intel.com file, replace content with its folder path on your machine.
  * Copy pushcontent@intel.com to your firefox extensions folder, for instance ~/.mozilla/firefox/cvvhptkl.default/extensions


Running the server
------------------

	cd server
	./cloudeebus.sh
